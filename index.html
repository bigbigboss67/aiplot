Import CIS & Leads from File

Supported File Formats

  • Excel (.xlsx, .xls) - Microsoft Excel files
  • CSV (.csv) - Comma-separated values
  • TXT (.txt) - Text files with tab/comma separation
  • PDF (.pdf) - PDF documents (text extraction)
  • Images (.png, .jpg, .jpeg) - Business cards, documents (OCR via Tesseract.js)
  • Documents (.doc, .docx) - Word documents (text extraction)
  • Website URL - Extract contact info from any webpage

Expected Columns

Your file should contain some of these columns (not all required):

Name, Phone, Email, Nationality, Budget, Location, Plot Type, Source, Notes

Auto-Detection

The system will automatically:

  • Detect column names
  • Create CIS records for all entries
  • Optionally create Leads from CIS records
  • Match data to available fields
  • Extract text from PDFs and images
